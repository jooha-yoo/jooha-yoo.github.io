AI Writing Tool

LLM-assisted writing tool that supports sustained goal-directed cognitive effort of human writers.
Academic NAACL 2025 paper published: https://digitalcommons.calvin.edu/calvin_facultypubs/984/

Problem

AI can seem helpful while still being inconsistent or hard to trust.

Built

An evaluation framework for finding reliability gaps in LLM behavior.

Impact

Clearer patterns for prompt design, system design, and human-AI evaluation.

Stride

AI-powered tool that turns daily voice logs into structured, traceable resume bullets by automatically synthesizing weekly reflections into interview-ready impact statements.

Problem

People struggle to recall and articulate their work over time, making it difficult to write accurate resume bullets or communicate internship impact.

Built

A voice-supported web app that captures daily reflections and uses AI to structure them into tasks, skills, and weekly synthesized summaries that generate resume bullets.

Impact

Enables users to capture work in real time and automatically produces traceable, source-linked resume bullets grounded in daily logs.

Centsible

A personal finance mobile app that turns spending data into clearer monthly insights.

Problem

Bank statements are hard to interpret when users want quick monthly insight.

Built

Transaction views, category reports, and an interactive donut chart.

Impact

Made spending habits easier to visualize and reflect on.

SSR and CSR Analysis

Explored and analyzed CSR and SSR rendering strategies through real-world implementation tradeoffs.

Problem

Developers often struggle to understand when to use client-side vs server-side rendering, especially when evaluating tradeoffs like performance, SEO, and interactivity in real applications..

Built

Researched and analyzed CSR vs SSR behavior using structured examples, showing how rendering strategies affect data flow, performance, and user experience.

Impact

Presented in front of 30+ classmates to help them understand move beyond definitions by connecting rendering strategies to real implementation tradeoffs in modern web applications.
